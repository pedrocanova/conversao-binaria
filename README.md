# Conversão Decimal para Binário

Programa simples em Python que converte números decimais para binário.

## Descrição

Este programa converte o número decimal **24015709** para sua representação binária usando o algoritmo de divisões sucessivas por 2.

## Como Executar

```bash
python conversao.py
```

## Resultado

O programa converte o número decimal 24015709 para binário:
- **Binário:** 1011011100111001101011101
- **Quantidade de bits:** 25

## Funcionalidades

- Implementação manual do algoritmo de conversão
- Verificação usando a função nativa `bin()` do Python
- Exibição formatada dos resultados
